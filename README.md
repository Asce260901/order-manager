# Order Management System

A JavaScript-based order management application that handles product inventory, customer orders, pricing, and order processing logic. Built with vanilla JavaScript to model a realistic business workflow.

## Features

- Create and manage customer orders
- Track product inventory and availability
- Apply discounts to order totals
- Process orders and update stock levels
- Display order summaries with itemized pricing

## Technologies Used

- HTML5
- JavaScript (ES6+)
- CSS3

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/Asce260901/order-management-system.git
```
2. Open `index.html` in your browser — no build step required.

## What I Learned

This project gave me hands-on experience modeling real-world business logic in JavaScript. I practiced working with arrays and objects to represent products and orders, writing functions to process data, and handling discount logic. It helped me understand how backend-like operations can be simulated on the frontend.
